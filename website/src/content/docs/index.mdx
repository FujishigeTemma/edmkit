## What you can do with edmkit

<CardGrid>
  <Card title="Forecast nonlinear systems" icon="rocket">
    Reconstruct an attractor from one observed series and predict its next state with simplex projection or S-Map.
  </Card>
  <Card title="Quantify nonlinearity" icon="puzzle">
    Sweep S-Map's locality parameter to measure how much prediction skill depends on local geometry.
  </Card>
  <Card title="Test causal direction" icon="approve-check-circle">
    Use Convergent Cross Mapping (CCM) to separate genuine coupling from shared trends and confounded correlation.
  </Card>
  <Card title="Keep analyses tractable" icon="random">
    NumPy-first APIs, a Rust-backed KDTree, batched operations, and an optional tinygrad GPU backend.
  </Card>
</CardGrid>
